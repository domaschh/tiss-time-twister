How does Tiss Time Twister work?

Let's get started

You can manage your TISS (or other) calendars by importing them via file or link.

After that, you can build your own configurations that override the calendar by creating one or more configurations that have certain rules.

Rules can either delete, modify, or tag an event in the calendar.

If you're confident that your configuration will also help others, feel free to publish it. Others can now clone this configuration and add it to their calendars. Imported configurations are cloned, so making changes will NOT affect the initial configuration. Additionally, there is also a set of default configurations provided for the TISS calendar specifically. You can find them on the public configurations page as well, together with their description.

After you have imported and modified a calendar, you can either export it via file (.ics) or generate a new link that you can simply use in your calendar app of choice. Beware that deleting configurations after you have imported the modified calendar into your app WILL have effects, meaning that when e.g. Google Calendar fetches the new improved calendar via URL again, the deleted configuration will NOT be used.

Why does this exist?

The TU Wien provides access to a calendar filled with lectures, exercises, exams, holidays, and more without any filters or regard to the event in question even existing. There is too much of it. Tiss Time Twister was created to fix that.

How do I use it?

The basic outline goes roughly like this:

import a calendar export the modified Calendar create/add Configurations enjoy less clutter

After that, you may find yourself occasionally returning to create new configurations or modify existing ones.

Importing a Calendar

Add a calendar to your Tiss Time Twister account to expose it to the modification pipeline.

You can add either a static file you wish to modify or a link to a calendar from some other source online.

  • TISS calendar
  • TUWEL calendar
Exporting a Calendar

Take your calendar, as it is after undergoing modifications, and export that as a URL in your calendar platform of choice to access it anywhere.

If you have created tags, you can select them, and the exported calendar will be filtered to include only events with that tag.

Creating a Configuration

Create a new configuration and apply it to a calendar.


What are Configurations?

Configurations are collections of Rules.

A Rule is a pair of a single Match and a single Effect.

The Match determines when a Rule's Effect is used. Blank fields are ignored. When all non-blank fields match with an Event, the Effect is applied to it.

The Effect is what the Rule does. It can either delete an Event outright, modify it, or tag it.

delete:
the event is not a part of the calendar anymore
modify:
replace title, description, or location - blank fields are ignored
tag:
apply a tag for filtering

If you are proud of one of your configurations and wish for others to enjoy it as well, you can set it to be public. After which, it shows up in the public configurations list, and others may add it to their calendars.

What are tags?

Tags are dynamic filters. When viewing calendars within Tiss Time Twister with tags selected or after exporting them with tags selected, the resulting calendar will only include events matching any of the tags.

Should you desire to export different courses separately, tags are a simple way of achieving exactly that.

Does the Tiss Time Twister only support calendars from TISS?

No. In theory, any .ics file or URL pointing at one should work. That said, .ics files may come with non-standard additions or formats that are not supported. Testing has only been done on calendars sourced from TISS and TUWEL, which are the intended use-case. Calendars from other sources might or might not work.

Can I import a calendar from Tiss Timer Twister?

Yes, as long as you do not create any circular references.

This can be useful if you want to change the order of operations. The default order of operations is:

default configurations delete/modify tags

Should you want to apply tags at the first step, simply import the calendar, set up your tagging, export the calendar, and import that URL for further modifications.

What does the Discord bot do?

It sends you regular reminders for deadlines or exam registration windows. All of its functions can be found by starting a message with a slash ("/") in the bot's Discord server and looking through the suggestions.
