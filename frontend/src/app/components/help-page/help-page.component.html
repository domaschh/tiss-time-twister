How does Tiss Time Twister work?

Let's get started

You can manage your TISS (or other) calendars by importing them via file or link.

After that you can build your own configurations that override the calendar by creating one or more configurations that have certain rules

Rules can either delete, modify or tag an event in the calendar.

If you're confident that your configuration will also help others. Feel free to publish it. Others can now clone this configuration and add it to their calendars. Imported configurations are cloned so making changes will NOT affect the initial configuration. Additionally there is also a set of default configurations provided for the TISS calendar specifically. You can find them on the public configurations page as well together with their description.

After you have imported and modified a calendar you can either export it via file (.ics) or generate a new link that you can simply use in your calendar app of choice. Beware that deleting configurations after you have imported the modified calendar into your app WILL have effects, meaning that when e.g Google Calendar fetches the new improved calendar via URL again the deleted configuration will NOT be used/
